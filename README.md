# Éloïse Lombard - Intégratrice web | Développeuse Frontend (Vue.js)

Intégratrice Web évoluant vers le développement Frontend (Vue.js), j'aime donner vie aux maquettes graphiques en interfaces soignées, accessibles et performantes.
J'accorde une grande importance à la qualité numérique ainsi qu'au parcours utilisateur qui sont au centre de mes préoccupations.
